🚀 5-MINUTE SETUP

1. Database
   createdb photohub_pro
   psql photohub_pro < backend/database/schema.sql

2. Backend
   cd backend
   cp .env.example .env
   [Edit .env with credentials]
   npm install
   npm run dev

3. Frontend (new terminal)
   cd frontend
   cp .env.example .env
   npm install
   npm run dev

4. Visit
   http://localhost:5173

⚙️ CREDENTIALS NEEDED

1. PostgreSQL Database
   - Local: psql (default)
   - Cloud: Neon, Railway, Heroku PostgreSQL

2. Cloudinary Account
   - Free: https://cloudinary.com (sign up)
   - Get: CLOUD_NAME, API_KEY, API_SECRET

3. Gmail Account
   - Enable 2-factor auth
   - Generate app password
   - Use in EMAIL_PASSWORD

4. JWT Secret
   - Any random string
   - Example: "your-random-secret-key-12345"

📊 TECH STACK

Backend:
  • Node.js + Express
  • PostgreSQL (database)
  • Cloudinary (image storage)
  • NodeMailer (email)
  • Node-Cron (scheduled tasks)
  • JWT + bcrypt (authentication)

Frontend:
  • React 18
  • Vite (bundler)
  • Tailwind CSS (styling)
  • Lucide React (icons)
  • React Router (routing)
