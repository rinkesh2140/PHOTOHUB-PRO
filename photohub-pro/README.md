# PhotoHub Pro v2.0 - Complete Photography SaaS Platform

Multi-tenant photography platform with all features included.

## Setup Instructions

### Backend
```bash
cd backend
npm install
createdb photohub_pro
psql photohub_pro < database/schema.sql
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Deployment
- Backend: Railway.app
- Frontend: Vercel.com
